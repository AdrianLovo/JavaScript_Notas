Sintaxis JS

  • Case Sensitive
  • Es de tipado dinamico
  • Comentarios en JS /* */ (Multilinea) o // (Linea)
  • Todas las sentencias deben terminar con ";" sin embargo no es estrictamente obligatorio Excepto en dos casos
    • Cuando se crea un objeto literal
    • Cuando se crea una funcion de expresiones
